The slider component is written in vanilla javascript with no component libraries. It will dynamically build as many slides as are made available in the HTML and create as many of the 'dots' below the text as necessary. The background image, all fonts, colors, sizes, etc. are completely customizable.
This is a great way to display 'Testimonials' for a business!
